# first game

## dependencies
python 3.13.5 (python.org)
pygame (pygamme.org)
git (git-scm.com)

## How to compile
open git CMD
use command git clone https://github.com/MeektyGames/first-game.git
open Command Prompt
type pip3 install pygame
go into the first-game folder
run the main.py file with python
